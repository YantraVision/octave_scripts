----------------------------------------------------------------------------
 This zip file provides the Matlab main script and functions for the simulation of Peak-to-Average Power Ratio (PAPR) reduction in MIMO OFDM/A system using block CMA algorithm. The package includes:
1) A main script called "MIMO_PAPR_MainScript".
2) Four m.function files which are used in the script.

instruction to use the code:
1) extract the zip file in a folder.
2) run the Matlab script named "MIMO_PAPR_MainScript".
3) one can change the parameters directly from the beginning of the "MIMO_PAPR_MainScript".
   file noted as "Define Parameters and Algorithms".
